This project focuses on performing end-to-end data analytics using Python. The notebook includes:

Data Collection & Dataset Understanding
Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
Monthly Sales Analysis
Customer & Product Insights
Data Visualization

The project uses a retail sales dataset stored in Excel format and analyzes sales performance, customer behavior, and product trends.

**Technologies Used**
Python
Jupyter Notebook
Pandas
NumPy
Matplotlib

 **Outcome**

* Successfully cleaned and analyzed retail sales data using Python.
* Performed Exploratory Data Analysis (EDA) to identify sales trends and customer insights.
* Generated meaningful visualizations for business decision-making.
* Improved understanding of data preprocessing, analytics, and visualization techniques.
* Developed hands-on experience with Pandas, NumPy, and Matplotlib.
* Extracted actionable insights from raw business data for performance evaluation.
